Welcome to CLEAN Database!

The CLEAN Database (CLEAN DB) is an open-access, comprehensive resource of high-accuracy enzyme function predictions. Powered by the in-house CLEAN AI tool, it has enabled an unprecedented expansion of enzyme function coverage based on the known protein sequence space. This release of CLEAN DB provides free access to over 55 million predicted enzyme functions spanning 1,098,305 organisms. By leveraging state-of-the-art protein language models (i.e., ESM v2), CLEAN DB also supports user-friendly prediction of mutation effects.
Learn more about the CLEAN Database below and get started using this powerful resource today to transform your research.
